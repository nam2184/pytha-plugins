# Pytha Plugins

This repository contains plugins for **Pytha 3D CAD** created using the [Pytha Lua API](https://github.com/pytha-3d-cad/pytha-lua-api).

## Overview

All plugins in this repository follow the **Pytha Lua API** guidelines for creating geometry, manipulating objects, and extending Pytha's functionality.

## Usage

1. Copy the desired plugin folder into your Pytha `plugins` directory.
2. Launch Pytha and access the plugin from the Plugins menu.
3. Follow any plugin-specific instructions provided in its Lua script.

## Contributing

Feel free to create new plugins or improve existing ones. Make sure to follow the [Pytha Lua API](https://github.com/pytha-3d-cad/pytha-lua-api) conventions.
